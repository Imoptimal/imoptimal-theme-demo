Theme color scheme represents warmth & luxury

Enabling users to build websites with ease, using some of the most popular website builders – Elementor, or Page Builder by SiteOrigin

You can find theme options within the Customizer

Here you’ll be able to choose the preferred way to build your website

Widgetization

This options enables you to build your website entirely through widgets (now interchangeable with blocks), providing different widget/block areas – header, page content, footer, sidebar (depending on the type of a page – homepage, single post page etc.).

Elementorization

This option removes the site header and footer, providing the user possibility to build out a whole page even with the free version of Elementor (only a pro version enables editing the page outside the content area).
